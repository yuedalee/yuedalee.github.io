## 一、先说明边界

这个仓库对应的是：

```text
悦达智科网站
```

它不是：

```text
悦达传媒官网
```

悦达传媒官网是甘肃悦达传媒有限公司的商业官网，部署在独立云端服务器上，用于承接影视传媒、宣传片、专题片、商业摄影、企业服务等业务。

悦达智科网站是悦达传媒围绕 AI 工具、商业方法论、Agent Skill、企业工作流和人机协同系统建设出来的公开入口。

两者有关联，但不能混为一个网站。

---

## 二、悦达智科当前定位

悦达智科当前对外定位为：

> 商业方法论 × AI Skill / Agent / 本地应用 / OA 协同系统

它不是一个提示词合集，也不是单纯展示 AI 工具的页面。

它更关注的是：

- 如何把企业经验沉淀成流程；
- 如何把商业方法论转译成 Agent Skill；
- 如何把重复任务、判断标准、沟通流程和复盘机制变成可执行的 AI 工作流；
- 如何让 AI 不只是聊天工具，而是进入内容、销售、管理、会议、知识库和组织协同系统；
- 如何让普通中小企业也能逐步建立自己的人机协同工作方式。

简单说：

> 悦达智科要做的不是“教人玩 AI 工具”，而是把企业经验、商业判断和工作流程，训练成可复用的 AI 生产力系统。
